https://nodejs.org/ko/

### Yarn 설치
과거에는 설치파일을 다운로드 받아서 설치했는데 npm 으로도 설치가 가능하다.
```bash
npm install --global yarn
```
다음 명령어로 설치가 제대로 됐는지 확인하자.
```bash
yarn --version
```
